# 🎧 Audio Novel App (Flutter)

A Flutter-based mobile application that provides an engaging audio storytelling experience.
Users can explore and listen to audio novels with a clean and simple interface.

---

## 🚀 Features

* 🎵 Audio-based novel experience
* 🔄 Smooth screen navigation using Navigator
* 🌐 Open external links using URL Launcher
* 📱 Clean and user-friendly UI
* ⚡ Fast and responsive performance

---

## 🛠️ Tech Stack

* Flutter
* Dart
* Material UI
* URL Launcher

---

## 📚 What I Learned

* Navigation in Flutter (Push, Pop, Routes)
* Using URL Launcher to open live websites
* Async & Await (handling asynchronous operations)
* Structuring a real-world mobile app
* UI design using Flutter widgets
